# ⚛️ React Basic Projects

A growing collection of beginner-to-intermediate **React game projects** built while leveling up frontend development skills.

This repository contains small but polished projects focused on **React state management, game logic, interactivity, UI design, and clean component structure.**

Think of this repo as a mini React arcade — and it’s only getting bigger.

---

# 🎮 Current Projects

## 🐍 Snake Game

A modern neon-styled arcade snake game.

### Features

- Smooth movement controls
- Food spawning system
- Collision detection
- Persistent high score (localStorage)
- Pause / Resume functionality
- Restart support
- Game-over overlay
- Modern glassmorphism UI

### Controls

| Key      | Action         |
| -------- | -------------- |
| ↑        | Move Up        |
| ↓        | Move Down      |
| ←        | Move Left      |
| →        | Move Right     |
| Spacebar | Pause / Resume |

---

## 🧠 Memory Game

A polished **2-player card-matching game**.

### Features

- Turn-based multiplayer logic
- Flip-and-match card system
- Dynamic score tracking
- Winner detection
- Restart system
- Animated winner overlay
- Glassmorphism card UI

### Gameplay

- Match pairs to score points
- Missed pair switches turn
- Highest score wins

# 🚀 Getting Started

Clone the repository:

```bash
git clone <your-repo-url>
cd React-basic-projects
```

Install dependencies for any project:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open:

```bash
http://localhost:5173
```
